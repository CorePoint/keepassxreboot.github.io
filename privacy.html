Privacy Policy

When visiting our website, we collect the following data:

  • Your anonymized IP address (the last two bytes are deleted)
  • Your browser user agent and plugin configuration
  • Your screen size configuration

The browser user agent usually contains the following information:

  • Your browser name and version
  • Your operating system name and version
  • Your processor architecture

All collected data are used for statistical purposes only and are not shared with any third parties. Statistics are generated using a self-hosted Piwik installation.

You can opt out of data collection at any time by clicking this opt-out link and following the instructions. Alternatively, you can also configure your browser to send a Do Not Track header.
