FAQ

Is Auto-Type supported on Mac OS X or Windows?
Yes, Auto-Type works cross-platform.

What password database formats are compatible with KeePassXC?
KeePassXC currently uses the KeePass 2 (.kdbx) password database format as the native format. It can also import KeePass 1 (.kdb) databases.

Why I get an error when I try to build from source for this Platform?
Please follow every step in the Wiki:Install-Instruction-from-Source page

Why on Ubuntu/Unity the tray icon, the tray menu and in-app shortcuts don't work?
This is a bug caused by the appmenu-qt5 package.
You have 3 option:

  1. Removing the appmenu-qt5 package
  2. Setting the Environment variable UBUNTU_MENUPROXY=''
  3. Setting the Environment variable QT_QPA_PLATFORMTHEME=''
Note By using the 1st or the 3rd, KeePassXC will lose the Unity look&feel.
